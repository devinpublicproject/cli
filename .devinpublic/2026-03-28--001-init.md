# Giới thiệu
- Đây là một dự án chạy trên command line
- Sử dụng Nodejs và typescript

# Yêu cầu phi chức năng
- Tôi muốn sử dụng package commander để làm command line interface, cho code clean và dễ maintain.
- Code ra phải clean, readable
- Tôi yêu cầu các code đều phải có testing, để đảm bảo code được maintain và không có lỗi.

# Kiến trúc dự án
- src: thư mục chứa source code
- tests: thư mục chứa các test cases
- src/cli.ts: file chính của dự án, chứa code cho command line interface
- src/actions: thư mục chứa các hàm xử lý logic của dự án
- src/utils: thư mục chứa các file tiện ích, như các file để xử lý arr, string, number,...

# Convention
- Tên file js, ts cần được đặt theo format 'ten-file.js,ts'. Dù là file class, file function, file test,... thì đều tuân theo convention này
- Với tên file nằm trong thư mục action, thì cần tuân theo format `<object>-<action>.ts`. Ví dụ `document-create.ts`
- Khi import, luôn phải import từ thư mục root
- Nếu được, thì cấu hình một alias là @ trỏ vào thư mục src

# Yêu cầu chức năng
- Dự án này cần được đóng gói thành package, để các dự án khác có thể cài đặt thông qua
```
npm install @devinpublic/dip
```

## Chức năng chính

1. Lệnh này sẽ tạo một file md trong thư mục .devinpublic

```
npm run dip
```

Khi chạy lệnh sẽ chỉ hỏi một câu đơn giản là
```
What are you about to do?
```

Sau khi user nhập và nhấn enter, lệnh sẽ tạo ra một file md trong thư mục .devinpublic. Đồng thòi in ra console một thông báo

```
File created successfully at .devinpublic/YYYY-MM-DD--001-init.md
```
